# 2605.09330 - The Trap of Trajectory: Spurious Correlations in Agentic Memory

- **Core contribution**: Identifies spurious correlations as a new vulnerability in agentic memory — retrieved memory carries miscorrelated evidence and propagates erroneous reasoning into downstream decisions. Proposes CAMEL calibration method.
- **Key concepts**: Spurious correlations through causal structure; trajectory-level memory contamination; CAMEL (plug-and-play calibration at write and retrieval time); adaptive attacks targeting calibration.
- **Method**: Benchmark canonical spurious patterns through causal structure, record across trajectory-level memory. CAMEL calibrates at both write and retrieval time, reducing spurious reliance while preserving clean performance.
- **Results**: Memory improves reasoning on clean inputs but amplifies reliance on spurious patterns when present. CAMEL consistently reduces spurious reliance across all three spurious pattern types, preserves/improves clean performance, robust under adaptive attacks.
- **CMD relevance**: Introduces a failure mode CMD's current label taxonomy doesn't cover — `spurious_correlation_error` or extends `reasoning_error`. Complementary to CMD: CAMEL prevents spurious patterns from entering memory; CMD detects when they already caused failure.
- **Open gap**: CAMEL requires causal structure identification; integration with CMD's counterfactual attribution pipeline unexplored.
