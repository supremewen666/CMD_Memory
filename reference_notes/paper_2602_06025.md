# BudgetMem Note

- arXiv: 2602.06025
- Core contribution: query-aware runtime budget-tier routing.
- Method sketch: memory modules expose low/mid/high tiers; a compact neural router selects tiers and is trained with RL under accuracy-cost tradeoff.
- Limitation: router does not own admission, graph association, or evolvable skill selection.
- Useful for new idea: per-query budget policy and cost-aware RL objective.
