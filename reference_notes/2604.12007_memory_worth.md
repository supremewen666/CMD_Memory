# Memory Worth: A Memory Governance Primitive

arXiv: 2604.12007 (2026-04)

## Core Contribution

Two-counter per-memory signal tracking co-occurrence with successful vs failed outcomes. Proves convergence to conditional success probability under stationary retrieval. Requires only two scalar counters per memory unit — no causal attribution needed.

## Key Results
- After 10,000 episodes: Spearman ρ=0.89 between MW and true utilities (vs ρ=0.00 static)
- Stale memories cross low-value threshold (MW=0.17); specialist memories stay high (MW=0.77)
- Supports staleness detection, retrieval suppression, deprecation without architectural changes
- Explicit limitation: MW measures association, not causation

## CMD Relevance

**Lightweight complement to CMD's heavyweight attribution.** Memory Worth's association signal (cheap, always-on) could serve as pre-filter for CMD's causal replay (expensive, on-demand). Two-tier architecture: MW monitors continuously (free), CMD replays on MW-triggered anomalies (expensive but precise).

Validates that useful memory governance signals can exist without causal attribution — CMD's causal attribution is the premium diagnostic tier above this baseline.
