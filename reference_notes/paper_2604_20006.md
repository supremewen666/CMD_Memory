# 2604.20006 - Memora: Benchmarking Long-Term Memory for Personalized Agents

- **arXiv**: 2604.20006
- **Venue**: ACL 2026 Findings
- **Core Contribution**: Memora benchmark spanning weeks-to-months conversations with three tasks (remembering, reasoning, recommending). Introduces FAMA (Forgetting-Aware Memory Accuracy) metric that penalizes reliance on obsolete memory.
- **Key Concepts**: Forgetting-aware evaluation, memory consolidation over time, obsolete memory penalty, personalized agent memory.
- **CMD Relevance**: Validates `item_stale` as a critical V1 label — current agents "frequently reuse invalid memories and fail to reconcile evolving memories." FAMA metric can serve as an external validation target for CMD's stale-item detection and repair.
- **Open Gap**: Memora detects stale memory use but does not diagnose why (extraction loss? retrieval failure? missing update?). CMD's counterfactual replay could attribute stale-memory failures to specific pipeline operations.
