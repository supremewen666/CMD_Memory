# 2605.14892 LIFE Survey: Failure Attribution in Multi-Agent Systems

- Core: First unified survey of collaboration + failure attribution + self-evolution in LLM-based multi-agent systems. 4 causally linked stages (LIFE framework). Error propagation across agents and interaction rounds is a key underexplored risk.
- Key: Existing surveys treat collaboration, failure attribution, and self-evolution separately — this is the first to examine causal dependencies among them.
- CMD relevance: Failure attribution recognized as distinct research subfield; multi-agent error propagation validates need for pipeline labels that track cross-agent memory contamination.
- Gap: Survey is taxonomic, not methodological. CMD provides the operational method for attribution that the survey calls for.
