# 2605.08715 AgentForesight: Online Auditing for Early Failure Prediction

- Core: Reframes failure attribution from post-hoc to online: at each step of an unfolding trajectory, auditor observes current prefix and must continue or alarm at earliest decisive error. Curates AFTraj-2K corpus across Coding and Web agent trajectories.
- Key: Online auditing enables intervention while trajectory is still unfolding (vs post-hoc which can only explain after failure). Auditor operates on prefix only, no future steps.
- CMD relevance: Attribution timing dimension: CMD is post-hoc (replay after failure); AgentForesight is online (detect before trajectory ends). Online attribution is a natural V2 extension for CMD: real-time replay triggering when online auditor flags anomaly.
- Gap: Detection only (alarm), no root-cause attribution or repair. CMD could provide the attribution backend when AgentForesight alarms.
