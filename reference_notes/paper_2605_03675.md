# MEMTIER Note

- arXiv: 2605.03675
- Core contribution: tiered memory architecture for long-running autonomous agents with episodic storage, retrieval weighting, consolidation, and adaptive retrieval policy.
- Key finding: memory coherence and retrieval bottlenecks matter over long operation windows.
- CMD relevance: provides a natural target system for `route_error`, `retrieval_error`, `compression_error`, and consolidation-related failure cases.
- Open gap: the paper reports system-level recall improvements but does not explain individual failures by memory operation.
