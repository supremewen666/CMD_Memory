# 2605.07313 - When Stored Evidence Stops Being Usable: Scale-Conditioned Evaluation of Agent Memory

- **Core contribution**: Scale-conditioned evaluation protocol — as irrelevant sessions accumulate, evidence usability degrades even when task evidence is held fixed. Introduces four diagnostics: budget-compliant reliability, tail memory-call burden, failure-regime decomposition, usable-scale boundary.
- **Key concepts**: Evidence-preserving growth protocol; usable-scale boundary (where reliability falls below target); interaction budget constraints; agent-dependent failure regimes.
- **Method**: Hold task evidence fixed while adding irrelevant sessions; log agent-memory trajectories; measure reliability degradation across scale.
- **Results**: On LongMemEval, HippoRAG loses 16-20pp budget-compliant reliability as irrelevant sessions added. LiCoMemory failures depend strongly on agent model (Qwen3-8B exceeds budget while Qwen3-32B/235B remain reliable). Reliability loss is not a single phenomenon.
- **CMD relevance**: Directly validates CMD's `retrieval_error` at scale — evidence exists in memory but becomes effectively irretrievable as irrelevant content accumulates. Supports CMD's counterfactual replay approach: replay must account for retrieval context size. The usable-scale boundary concept maps to CMD's attribution threshold.
- **Open gap**: No automated method to determine optimal retrieval budget or to detect when scale has crossed usable boundary.
