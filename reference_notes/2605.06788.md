# 2605.06788 Conformal Agent Error Attribution

- Core: Conformal prediction for error attribution in multi-agent systems; finite-sample distribution-free coverage guarantees; contiguous sequence predictions for efficient recovery.
- Key: Filtration-based CP algorithms for sequential data (agent trajectories); predicts contiguous error sets (not arbitrary subsets); enables efficient recovery and debugging.
- CMD relevance: Formal statistical framework for attribution confidence that CMD currently lacks. Conformal sets could bound CMD's attribution uncertainty. Contiguous predictions align with CMD's top-2 attribution.
- Gap: Observational (trajectory-based), not counterfactual. Provides coverage guarantees but not causal evidence. Complementary to CMD: ConformalAttr for confidence bounds, CMD for causal mechanism.
