# MemMachine: Ground-Truth-Preserving Memory Note

- arXiv: 2604.04853
- Core contribution: open-source memory system integrating short-term, long-term episodic, and profile memory while preserving complete conversation turns; 93.0% accuracy on LongMemEval with ~80% token reduction vs Mem0.
- Key concepts: ground-truth preservation, three-tier memory (STM/LTM-episodic/profile), complete conversation turn retention, token efficiency.
- CMD relevance: MemMachine's ground-truth preservation goal aligns with CMD's Verbatim Event Oracle — both recognize that losing raw evidence during memory extraction is a critical failure mode. The 80% token reduction vs Mem0 validates CMD's compression_error label.
- Open gap: ground-truth preservation is an architectural choice, not a diagnostic framework; CMD detects when preservation fails.
