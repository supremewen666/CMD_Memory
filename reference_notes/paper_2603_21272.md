# Library Theorem: External Memory Bounds Note

- arXiv: 2603.21272
- Core contribution: formal proof that tool-augmented agents with indexed external memory achieve O(log_b N) retrieval vs Ω(N) for unindexed, with experiments showing median 1-page reads for abstract content.
- Key concepts: indexed external memory, exponential retrieval cost reduction, O(log_b N) bounds, formal proof.
- CMD relevance: provides theoretical grounding for why retrieval quality matters in agent memory systems. The formal bounds justify CMD's focus on retrieval_error as a first-class pipeline label.
- Open gap: the theorem covers retrieval cost, not retrieval correctness; CMD addresses whether the right content was retrieved, not just how efficiently.
