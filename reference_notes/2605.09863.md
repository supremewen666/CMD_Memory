# 2605.09863 - Nautilus Compass: Black-box Persona Drift Detection for Production LLM Agents

- **Core contribution**: Black-box persona drift detector operating entirely at prompt-text layer (no LLM calls at index time). Only public agent memory layer that embeds raw conversation text directly without extraction. ROC AUC 0.83 for drift detection.
- **Key concepts**: Persona drift (forgetting constraints, repeating flagged mistakes, confabulating agreements); BGE-m3 cosine similarity between user prompts and behavioral anchors; Merkle-chained audit log; no-extraction design.
- **Method**: Raw conversation text embedded directly; drift detected via weighted top-k mean cosine similarity; ships as Claude Code plugin + MCP A2A server + CLI + REST API.
- **Results**: ROC AUC 0.83 drift detection; LongMemEval-S 56.6% (~30 points below white-box leaders, architectural ceiling of no-extraction design); EverMemBench-Dynamic 44.4% (tops 4 published baselines); $3.50 reproduction cost.
- **CMD relevance**: Complementary to CMD — Compass detects drift (behavioral deviation), CMD attributes memory failure causes. Compass's no-extraction design validates that raw text can serve as anchor without LLM preprocessing. Potential integration: Compass as drift monitor → CMD as attribution engine when drift detected.
- **Open gap**: No-extraction ceiling at 56.6% LongMemEval — tradeoff between cost and precision. Does not diagnose why drift occurred.
