# What Happens Inside Agent Memory? Note

- arXiv: 2605.03354
- Core contribution: analyzes internal memory circuits from emergence to diagnosis across model scales and memory frameworks.
- Key finding: routing/control signals can appear earlier than content extraction or grounding signals, creating silent failure modes where agents route plausibly but fail to retain or ground information.
- Diagnostic signal: feature-space separation reportedly supports operation-level failure localization.
- CMD relevance: add internal-signal diagnosis as a future optional feature, but keep black-box counterfactual replay as the primary method.
