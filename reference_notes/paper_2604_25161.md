# 2604.25161 - Capability-Oriented Failure Attribution for VLN Agents

- **arXiv**: 2604.25161
- **Venue**: ACL 2026
- **Core Contribution**: Capability-oriented testing framework that attributes VLN agent failures to specific capability deficiencies (perception, memory, planning, decision) using capability oracles and feedback-driven test generation.
- **Key Concepts**: Capability oracle, adaptive test generation, failure-to-capability attribution, memory as a testable capability.
- **CMD Relevance**: Validates failure attribution as an emerging research trend. Capability oracles are coarser-grained analogues of CMD's replay portfolio (capability-level vs operation-level). Memory is explicitly treated as a diagnosable capability.
- **Open Gap**: Uses observational oracles, not counterfactual intervention. Attribution is capability-level, not operation-level (cannot distinguish write_error vs retrieval_error within the memory capability).
