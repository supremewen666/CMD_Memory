# 2605.13438 - Cognifold: Always-On Proactive Memory via Cognitive Folding

- **Core contribution**: Brain-inspired "always-on" agent memory extending Complementary Learning Systems (CLS) from 2 layers to 3 (adding prefrontal intent layer). Continuously folds fragmented event streams into self-emerging cognitive structures.
- **Key concepts**: Three-layer CLS (hippocampus, neocortex, prefrontal intent); graph-topology self-organization; cognitive structures that merge/split/decay/relink; intent surfacing when concept-cluster density crosses threshold.
- **Method**: Proactive structural formation under event stream; semantic merging of similar structures; temporal decay for stale; associative recall for relinking; intent detection via cluster density threshold.
- **Results**: CogniFold uniquely produces memory structures matching cognitive expectations on CogEval-Bench. Performs robustly across 7 benchmarks spanning 5 cognitive domains.
- **CMD relevance**: Proactive memory organization is orthogonal to CMD's reactive failure diagnosis — CogniFold could reduce failures CMD is designed to detect. The intent-layer concept could inform CMD's Failure Memory context construction (what intent was the memory serving?). The decay mechanism relates to `item_stale` (V1 label).
- **Open gap**: No failure diagnosis capability — when proactive organization fails, how to detect and attribute?
