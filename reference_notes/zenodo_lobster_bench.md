# LOBSTER-Bench: Long-Lived Agent Observability Benchmark

- Core: First benchmark for long-lived agent observability. 6 dimensions: temporal persistence, cognitive telemetry coverage, relational observability, collective task assay, cognitive load management, governance & auditability.
- Key: Real data from 21-agent system over 7 days: 27,788 telemetry rows, 6,354 settled wagers, 2,546 template-violation events. Emergent cascade failure: two individually correct subsystems composed into serial degradation of ten agents under 30-second mutual-templating signal. "Temporal depth is not elapsed runtime. It is the unfakeable trace of observation, revision, and emergent failure."
- CMD relevance: Validates CMD's core premise — persistent agents need memory observability. Cascade failure is exactly the multi-operation failure pattern CMD diagnoses. Governance & auditability dimension maps to Post-Repair Context Replay as audit trail. Complementary: LOBSTER-Bench measures whether a system CAN be observed; CMD provides WHAT to observe when it fails.
- Gap: LOBSTER-Bench defines the benchmark dimensions but doesn't provide attribution mechanisms. CMD fills this gap with counterfactual replay for operation-level diagnosis.
