# MedEinst: Counterfactual Diagnosis Benchmark Note

- arXiv: 2601.06636
- Core contribution: counterfactual benchmark with 5,383 paired clinical cases; proposes ECR-Agent with Dynamic Causal Inference (association/intervention/counterfactual levels) and Critic-Driven Graph and Memory Evolution (CGME).
- Key concepts: three-level causal reasoning (association, intervention, counterfactual), CGME iterative memory refinement, evidence audit, bias trap rates in frontier LLMs.
- CMD relevance: MedEinst's three-level causal hierarchy validates CMD's counterfactual approach — intervention-based replay (level 2) and counterfactual attribution (level 3) are distinct from observational diagnosis (level 1). CGME's validated reasoning path storage parallels CMD's Failure Memory.
- Open gap: MedEinst's counterfactuals are dataset-level paired cases, not operation-level memory replay; CMD applies counterfactual logic to the memory pipeline itself.
