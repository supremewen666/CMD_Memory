# MemGPT: OS-Inspired Virtual Context Note

- arXiv: 2310.08560
- Core contribution: foundational virtual context management for LLMs, inspired by OS hierarchical memory; intelligently manages memory tiers to provide extended context within fixed context windows.
- Key concepts: virtual context management, memory tiering (fast/slow), interrupt-driven control flow, document analysis beyond context window, multi-session conversational agents.
- CMD relevance: MemGPT established the memory-tier paradigm that CMD's target agents inherit. CMD's pipeline labels (write, compression, retrieval) directly correspond to MemGPT's tier management operations. MemGPT's interrupt mechanism is an early form of the monitor signal CMD uses.
- Open gap: MemGPT focuses on extending context, not diagnosing failures; CMD addresses what happens when tier management operations fail.
