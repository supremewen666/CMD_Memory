# ByteRover: Agent-Native Hierarchical Memory Note

- arXiv: 2604.01599
- Core contribution: agent-native memory where the same LLM curates, structures, and retrieves knowledge; uses hierarchical Context Tree with 5-level progressive retrieval, no vector DB or graph DB dependency.
- Key concepts: agent-native curation, hierarchical Context Tree, 5-level progressive retrieval, zero external DB dependency.
- CMD relevance: ByteRover's "same LLM curates and retrieves" design means write/extraction/retrieval errors are all caused by the same model — this tight coupling is exactly the failure mode CMD is designed to disentangle through counterfactual replay.
- Open gap: no failure diagnosis when the LLM's curation introduces errors; CMD's pipeline-stage isolation through replay directly addresses this.
