# Storage Is Not Memory Note

- arXiv: 2605.04897
- Core contribution: argues for retrieval-centered agent recall over verbatim preserved events rather than extraction-first memory schemas.
- Key claim: content discarded at ingestion cannot be recovered once the future query is known.
- CMD relevance: introduces a likely failure mode for CMD: premature abstraction or extraction loss before retrieval.
- Method implication: add a Verbatim Event Oracle that replays from raw preserved events to distinguish ingestion loss from retrieval failure.
