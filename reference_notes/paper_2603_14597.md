# D-MEM: Dopamine-Gated Agentic Memory Note

- arXiv: 2603.14597
- Core contribution: biologically inspired Fast/Slow memory routing via Reward Prediction Error gating; a Critic Router scores stimuli for Surprise and Utility, only high-RPE inputs trigger O(N) knowledge graph restructuring.
- Key concepts: Critic Router, RPE-gated dopamine signal, O(1) fast buffer vs O(N) slow evolution, LoCoMo-Noise benchmark, 80% token reduction, eliminates O(N²) write-latency.
- CMD relevance: the Critic Router's Surprise/Utility scoring is a lightweight anomaly detection mechanism that could serve as a cheap pre-filter before expensive CMD counterfactual replay, similar to how Subagent Judge Monitor gates replay.
- Open gap: D-MEM detects surprise but does not attribute to specific memory operations; CMD's operation-level attribution complements this architecture.
