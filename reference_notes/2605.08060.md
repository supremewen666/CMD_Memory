# 2605.08060 The Memory Curse: Expanded Recall Erodes Cooperative Intent

- Core: Context window expansion systematically degrades cooperation in multi-agent social dilemmas. 18/28 model-game settings show cooperation drop. Mechanism: eroding forward-looking intent (not rising paranoia). LoRA on forward-looking traces mitigates and transfers zero-shot.
- Key: 378,000 reasoning traces analyzed. Memory sanitization (replacing visible history with random but structurally identical history) partially restores cooperation. Memory has negative social externalities beyond correctness.
- CMD relevance: Memory side effects — expanded recall can harm cooperation even when memories are "correct." CMD's repair evaluation should consider multi-dimensional impact (correctness + cooperation + security), not just single-task accuracy.
- Gap: Identifies the problem; doesn't propose memory repair or filtering. CMD could extend repair to filter cooperation-degrading memories.
