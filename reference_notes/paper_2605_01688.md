# GRAVITY: Structured Anchoring for Conversational Memory Note

- arXiv: 2605.01688
- Core contribution: plug-and-play structured memory module extracting entity profiles, temporal event tuples, and cross-session topic summaries; 7.5-10.1% judge accuracy improvement on LongMemEval and LoCoMo.
- Key concepts: entity profiles, temporal event tuples, cross-session topic summaries, structured anchoring, context injection.
- CMD relevance: GRAVITY's structured extraction (entity profiles, event tuples) is a concrete instance of the write/extraction pipeline CMD audits. If entity extraction fails, CMD would attribute it as write_error or premature_extraction_error.
- Open gap: no mechanism to detect when structured extraction silently drops critical information; CMD's Verbatim Event Oracle catches this.
