# 2605.10870 - Remember the Decision, Not the Description: A Rate-Distortion Framework for Agent Memory

- **Core contribution**: Reframes agent memory compression as decision-centric rate-distortion: memory quality = loss in achievable decision quality induced by compression, not description fidelity. Derives exact forgetting boundary and memory-distortion frontier.
- **Key concepts**: Decision-centric memory; rate-distortion framework; forgetting boundary; DeMem (online memory learner that refines partitions only when data certify decision conflict); near-minimax regret guarantees.
- **Method**: Measure memory by loss in decision quality under compression budget. DeMem refines its partition only when a shared state would induce decision conflict — merges states that lead to same decisions, splits those that don't.
- **Results**: DeMem yields consistent gains under same runtime budget on synthetic diagnostics and long-horizon conversational benchmarks.
- **CMD relevance**: Challenges CMD's `compression_error` framing — suggests compression is not inherently harmful, only harmful when it collapses decision-relevant distinctions. CMD's counterfactual replay could incorporate decision-centric quality measure. DeMem's partition-refinement logic parallels CMD's operation-level attribution granularity.
- **Open gap**: Framework assumes known decision space; CMD operates on open-ended agent tasks where decisions are implicit.
