# 2605.14865 Holistic Evaluation and Failure Diagnosis of AI Agents

- Core: Pairs top-down agent-level diagnosis with bottom-up span-level evaluation; decomposes analysis into independent per-span assessments that scale to arbitrary trace length; produces span-level rationales for each verdict.
- Key: SOTA on TRAIL benchmark (GAIA + SWE-Bench), 38% relative gain on category F1, up to 3.5x on localization over strongest prior baselines.
- CMD relevance: Closest adjacent work to CMD's attribution — span-level diagnosis is coarser than operation-level; observational (not counterfactual) evidence; no repair validation loop.
- Gap: Span-level tells you which span failed, not which memory operation caused the failure. CMD's operation-level Recovery Gain is more actionable for memory pipeline debugging.
