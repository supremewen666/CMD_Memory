# 2605.08717 PROBE: Debugging the Debuggers — Failure-Anchored Structured Recovery

- Core: Organizes failed-run telemetry into structured evidence → structured diagnosis → bounded recovery guidance. Telemetry Layer preserves fine-grained runtime signals; Diagnosis Layer fuses cross-signal evidence; Guidance Gate produces diagnosis-driven bounded recovery.
- Key: Converts heterogeneous runtime evidence into grounded, bounded recovery guidance. Domain: software engineering agents.
- CMD relevance: Same detect→diagnose→guide pattern as CMD. PROBE for SE agent runtime telemetry; CMD for memory pipeline operations. Both produce bounded (not open-ended) recovery guidance.
- Gap: PROBE uses observational telemetry; CMD uses counterfactual replay. Different domains, same architectural pattern — mutual validation.
