# AgeMem Note

- arXiv: 2601.01885
- Core contribution: STM/LTM memory operations become tool actions learned inside the agent policy.
- Method sketch: six tools cover summarize, clear/filter, retrieve, add, update, delete; three-stage RFT teaches write, context control, then integrated QA.
- Limitation: policy learning is not coupled to graph-structured associative retrieval.
- Useful for new idea: action-space design and curriculum.
