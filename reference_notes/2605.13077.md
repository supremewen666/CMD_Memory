# 2605.13077 Counterfactual Reasoning for Causal Responsibility Attribution

- Core: Models multi-agent systems as concurrent stochastic multi-player games; retrospective (backward) counterfactual responsibility using Shapley value; formal verification + strategic reasoning.
- Key: Shapley value satisfies fairness and consistency axioms; framework supports both verification and strategic reasoning in responsibility-aware MAS. Nested fixed-point logic for probabilistic verification.
- CMD relevance: **Closest formal work to CMD's counterfactual approach.** Key difference: Shapley value measures average marginal contribution across all agent coalitions (exponential); CMD's Recovery Gain measures single-operation intervention effect (linear in pipeline length). Shapley is coalitional; CMD is operation-sequential.
- Gap: Shapley could complement CMD for multi-agent extension (allocate responsibility across agents, then within-agent attribution across pipeline operations). Combined approach: Shapley for inter-agent + Recovery Gain for intra-agent.
