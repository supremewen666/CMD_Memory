# 2604.20117 - SCG-MEM: Schema-Constrained Generation for Agent Memory

- **arXiv**: 2604.20117
- **Core Contribution**: Memory access reframed as Schema-Constrained Generation — LLM decoding constrained by dynamic Cognitive Schema to generate only valid memory keys (formal guarantee against "structural hallucination"). Updates follow Piagetian assimilation/accommodation. Associative Graph enables multi-hop reasoning.
- **Key Concepts**: Schema-constrained generation, structural hallucination, cognitive schema, assimilation/accommodation, associative graph.
- **CMD Relevance**: Introduces "structural hallucination" — a failure mode CMD's V0 label set does not cover (model invents non-existent memory keys). This suggests a potential V2 label or a subtype of `reasoning_error`. SCG-MEM's schema as ground truth for valid memory access is analogous to CMD's gold evidence as ground truth for attribution.
- **Open Gap**: Schema constrains generation but does not diagnose which memory operation produced the wrong key. CMD could attribute structural hallucination to retrieval route vs generation failure.
