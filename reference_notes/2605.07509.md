# 2605.07509 MASPrism: Lightweight Failure Attribution

- Core: Failure attribution using prefill-stage signals (token-level NLL + attention weights) from a small language model; no decoding needed; reconstructs focused diagnostic prompt.
- Key: Symptom-like steps identified via token-level signals; candidate source steps traced via attention weights; lightweight (prefill only, no generation).
- CMD relevance: Cheapest attribution alternative (no replay, no generation). Observational/heuristic approach vs CMD's counterfactual/causal. Could serve as cheap pre-filter before expensive CMD replay.
- Gap: Signal-based = correlational; counterfactual replay = causal. MASPrism tells you which step looks anomalous; CMD tells you which operation, if corrected, would recover the answer.
