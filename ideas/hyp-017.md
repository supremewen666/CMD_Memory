# Hypothesis 017

## Hypothesis
Multi-Resolution Counterfactual Attribution: The agent failure attribution subfield is self-organizing into a granularity spectrum (chunk → agent → operation), with counterfactual intervention as the common methodological thread across independently developed systems.

## Reasoning

Day 7 metabolism found three independently developed systems using counterfactual approaches at different levels of granularity: TraceAudit (chunk-level counterfactual auditing of RAG), VerifyMAS (agent-level hypothesis verification for multi-agent attribution), and CMD (operation-level counterfactual replay for memory pipeline diagnosis). All three emerged within the same 2-week window (May 2026), suggesting the field is converging on counterfactual as the right foundation for agent failure attribution.

The granularity spectrum is organizing naturally: chunk-level (TraceAudit: "which retrieved chunk caused the error") → agent-level (VerifyMAS: "which agent in the MAS caused the failure") → operation-level (CMD: "which memory pipeline operation caused the failure"). No existing system spans multiple levels, and the levels are complementary rather than competing — a chunk error could be caused by an agent error, which could be caused by a memory operation error.

Additionally, trace-mem introduces a fourth counterfactual paradigm: preventive counterfactual gating at ingestion time ("admit only if accuracy improves"). This complements the three retrospective approaches, suggesting a full spectrum: preventive-gate → chunk-audit → agent-attribution → operation-diagnosis.

The fact that these systems were developed independently (TraceAudit at AAAI target, VerifyMAS at multi-agent venue, CMD at memory diagnosis) indicates the counterfactual approach is not an isolated innovation but a converging paradigm.
