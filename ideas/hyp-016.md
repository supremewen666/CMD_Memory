# Hypothesis 016

## 假设
CMD's operation-level counterfactual attribution with Recovery Gain is a distinct and defensible contribution relative to the four new attribution paradigms that emerged in May 2026: Shapley-value counterfactual (2605.13077), conformal prediction (2605.06788), prefill-stage signals (2605.07509), and span-level diagnosis (2605.14865). The paper must explicitly differentiate against all four.

## 推理过程

In a single week (2026-05-10 to 2026-05-15), 5+ papers proposed distinct failure attribution methods for agent systems. This is the fastest-growing subtopic in agent memory research.

2605.13077 (Counterfactual Responsibility Attribution) is the closest formal work: it uses Shapley value to allocate responsibility among agents in concurrent stochastic games. Shapley value measures each agent's average marginal contribution across all coalitions. CMD's Recovery Gain (Δk) measures single-operation recovery: Δk = Metric(ŷ_k, y) - Metric(ŷ, y), where k ranges over pipeline operations (write, compress, extract, retrieve, inject, route, reason). The key difference: Shapley is coalitional (requires evaluating all subsets, exponential cost), while Recovery Gain is operation-sequential (linear in pipeline length). For memory pipeline debugging, the linear approach is more practical and maps directly to actionable repairs.

2605.14865 (Holistic Eval) does observational span assessment at span-level granularity. CMD does intervention-based assessment at operation-level granularity. Operation-level is finer and maps to specific pipeline fixes.

2605.06788 (ConformalAttr) provides statistical coverage guarantees but uses observational trajectory evidence. CMD uses counterfactual intervention evidence, which provides causal rather than correlational attribution.

2605.07509 (MASPrism) is the cheapest approach (prefill signals, no decoding) but relies on heuristic token-level signals. CMD's replay is more expensive but produces causal evidence.

The paper's contribution claim should rest on: (1) operation-level granularity is uniquely actionable for memory pipeline debugging, (2) counterfactual intervention evidence (Recovery Gain) is causally grounded where observational methods are correlational, (3) Post-Repair Context Replay provides automated semantic validation that none of the four alternatives offer.
